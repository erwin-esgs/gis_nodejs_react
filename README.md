This project is frontend + backend of : https://github.com/erwin-esgs/ELK-docker-compose , you can start the docker first.
# Start the backend
## 1.Install

### npm

```
npm i
or
npm i --legacy-peer-deps
```

### yarn

```
yarn install
```

## 2.Start

```sh
npm start
or
yarn start
```
